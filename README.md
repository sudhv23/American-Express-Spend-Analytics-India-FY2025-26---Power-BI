# 💳 American Express Spend Analytics – India (FY2025–26)

## 📊 Overview

This project showcases a **Power BI dashboard** analyzing spending behavior of **American Express (AmEx) cardholders in India**.

The dashboard is designed to simulate how analytics teams in the **FinTech / Payments industry** derive insights from transaction data to support business decisions.

---

## 🎯 Objectives

- Analyze **customer spending patterns**
- Identify **top merchant categories and cities**
- Evaluate **average transaction value (ticket size)**
- Understand **customer segmentation**
- Provide **market context** using credit card network share in India

---

## 📌 Key Metrics (KPIs)

- **Total Spend (₹ Cr)**
- **Total Transactions**
- **Active Cardholders**
- **Average Transaction Value (₹)**
- **Average Monthly Spend per Customer (₹)**
- **Top Merchant Category**

---

## 📈 Dashboard Visuals

- **Spend Trend (Monthly)**
- **Top Merchant Categories by Spend**
- **Average Transaction Value by Category**
- **Top Cities by Spend**
- **Spend by Customer Segment**
- **Credit Card Network Share – India (FY2025–26)**

---

## 🧠 Key Insights

- American Express holds a **small market share (~2%)** in India but exhibits **higher average transaction value**
- **Travel and hospitality** categories dominate spending
- **Metro cities** contribute the majority of transaction value
- **Premium and affluent segments** drive most of the spend
- Demonstrates a **low-volume, high-value transaction model**

---

## 🗂️ Dataset Information

- The dataset used in this project is an **AI-generated / AI-simulated dataset**
- Created solely for **educational and demonstration purposes**
- Designed to mimic realistic transaction behavior and analytical scenarios

### ⚠️ Disclaimer

> This dataset does **not represent actual American Express data**.  
> It is intended to simulate real-world analytics use cases.

- The **credit card network share** reflects industry-level estimates  
- The **transaction dataset is a scaled sample**, not actual market volumes  
- The focus is on **behavioral insights**, not precise financial figures
